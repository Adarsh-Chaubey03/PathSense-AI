# PathSense-AI

PathSense-AI is an intelligent assistive system designed to enhance safety for visually impaired individuals by detecting falls and providing immediate emergency support. The system leverages smartphone-based IMU sensors (accelerometer and gyroscope) to continuously monitor motion, identify fall events accurately, and trigger real-time SOS alerts with location sharing to pre-configured contacts.

The solution focuses on minimal user interaction and maximum accessibility by integrating voice feedback and haptic alerts. PathSense-AI aims to reduce response time during critical situations, improving user safety, independence, and confidence in everyday life through reliable, real-time monitoring and assistance.
